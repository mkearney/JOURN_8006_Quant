Measures of central tendency

Central tendency

  • These statistics describe the center (middle) of a distribution:
  • Mode
  • Median
  • Mean
  • Which measure is best depends on data type and context.

Mode

  • Mode describes the most common data point.
  • The value, response, category, etc. that occurs most frequently.
  • Particularly useful when describing categorical, ordinal, or non-normal (skewed) integer data.
  • Not useful when dealing with normal, continuous data.

\[ x = 5.1465, 8.2413, 3.9485, 3.2934 \]

Calculating the mode

  • To find the mode, create a frequency table (two columns).
  • List all possible values on one side (column) and keep a tally (count) of observations for each value on the right side.
  • Value with the highest count is the mode > And, yes, it’s possible to have multiple modes

Median

  • Median describes the mid-point of a distribution.
  • The value found at the center (50%) of the data.
  • Particularly useful when describing—esp skewed—distributions.

Calculating the median

  • To find the median, create a string of N values ordered from low-to-high or high-to-low.
  • If N is odd, report the one middle value (it should be equidistant in the number of observations on either side)
  • If N is even, find the two middle values and report their average

Mean

  • Mean is the expected value of any given observation assuming no other information is provided.
    • Example: let’s say the mean height of KU students is 5’7’’. If we randomly selected one KU student, what would you predict their height to be?
  • We typically think of the mean when we use the word “average.”

Calculating the mean

  • To calculate the mean of x, sum (add together) all values of x and divide by N (the number of observations). \[ \mu = \frac{\sum x}{N} \]

Why know the mean?

  • Because it’s the best predictor of any given observation
  • Guessing the mean (instead of the median or mode, for example) should result in the smallest amount of error
    • Error here refers to degree to which an observed value differs from an expected value.

A mean[ingful] example

  • One last example: Let’s say your goal is to get as close as possible to predicting the outcome of a dice roll (two 6-sided dice)
  • If reducing the amount of error is any consolation, then guess the mean
  • If the only path to victory is guessing the exact outcome, then guess your favorite number.
    • Dice roller: https://www.random.org/dice/

Measures of dispersion

Range

  • Range desribes the distance between a distribution’s maximum and minimum values
  • Maximum is the highest/largest observed value
  • Minimum is the lowest/smallest observed value

\[ range = min - max \]

  • The range is often reported along with the maximum and minimum, e.g, the sample consisted of exclusively undergraduates (M age = 21.20, SD = 2.55, range = 18 - 24).

Variability

  • Variance refers to the degree to which a distribution varies
  • It’s measured by calculating the distance from the mean for each observation
  • When variance is low, there is little change in data values.
  • When variance is high, there is lots of change in data values.

Standard deviation

  • The standardized form of variance is referred to as the standard deviation
  • Standard deviations describe the average distance from the mean.
  • The mean can be found at zero standard deviations from the mean, or the 50% percentile
  • For a normal distribution, 95% of the observations should fall within plus or minus 2 standard deviations

Calculating standard deviation

  • Don’t worry about memorizing this, but it’s useful to see how it works

\[ s^2 = \sqrt{ \frac{\sum x - \bar{x}}{N - 1}} \]

\[ sd = \sqrt{s^2} \]

Easier standard deviation formula

  • There’s a shortcut formula that is useful to know:

\[ s^2 = \sqrt{\frac{\sum X^2 - \frac{(\sum X)^2}{N}}{N - 1}} \]
